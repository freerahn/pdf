# PDF 편집 도구

웹 브라우저에서 PDF 파일을 편집할 수 있는 도구입니다.

## 주요 기능

- 📁 **PDF 파일 업로드**: 로컬 PC에서 PDF 파일을 불러옵니다
- ✂️ **PDF 분할**: 원하는 페이지 범위를 선택하여 새로운 PDF로 분할합니다
- ✏️ **텍스트 편집**: 특정 페이지에 텍스트를 추가할 수 있습니다
- 🗑️ **페이지 삭제**: 불필요한 페이지를 삭제합니다
- 💾 **PDF 다운로드**: 편집된 PDF를 다운로드합니다

## 사용 방법

1. **프로젝트 실행**
   ```bash
   npm install
   npm start
   ```
   또는 간단히 `index.html` 파일을 브라우저에서 직접 열어도 됩니다.

2. **PDF 파일 업로드**
   - "PDF 파일 선택" 버튼을 클릭하여 로컬 PC에서 PDF 파일을 선택합니다

3. **PDF 분할**
   - 시작 페이지와 끝 페이지를 입력하고 "분할 실행" 버튼을 클릭합니다

4. **텍스트 편집**
   - 페이지 번호, 텍스트 내용, X/Y 좌표, 글자 크기를 입력하고 "텍스트 추가" 버튼을 클릭합니다

5. **페이지 삭제**
   - 삭제할 페이지 번호를 입력하고 "페이지 삭제" 버튼을 클릭합니다

6. **다운로드**
   - 편집이 완료되면 "PDF 다운로드" 버튼을 클릭하여 편집된 PDF를 저장합니다

## 기술 스택

- **PDF.js**: PDF 파일 렌더링 및 미리보기
- **pdf-lib**: PDF 편집 및 조작
- HTML5, CSS3, JavaScript

## 주의사항

- 브라우저 호환성: 최신 버전의 Chrome, Firefox, Edge, Safari에서 작동합니다
- 큰 PDF 파일의 경우 처리 시간이 걸릴 수 있습니다
- 텍스트 편집은 텍스트 추가만 가능하며, 기존 텍스트 수정은 지원하지 않습니다
